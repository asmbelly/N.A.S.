# N.A.S.
A networking auto configuration tool and WiFi stabilizer, built to adjust DNS providers to enhance system usability and network ping.

N.A.S. monitors system DNS, Internet, and WiFi usage and capabilitites and adjusts providers to lower ping.

_____________________

## INSTALLATION

### REQUIREMENTS

- Python 3.10 or Newer

  Yea that's all
----------------

Download the ZIP file from the releases page and extract it, then run this text in the command prompt:
```Powershell
cd network-auto-stabilizer
python main.py
```

The command prompt gets immediately cleared and it should show some text identifiying your network ping, packet loss, Wifi signal, Status, and current DNS service. Your DNS service should update every ~3 seconds, the automation is trying to lower your network ping.
